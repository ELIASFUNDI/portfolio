Amsterdam NDVI Analysis

2024  ·  Vegetation Index  ·  ArcGIS Pro · Raster Analysis · Remote Sensing

Normalised Difference Vegetation Index (NDVI) analysis of Amsterdam, Netherlands, mapping the spatial distribution of green space across the city using multispectral satellite imagery. The output reveals urban green cover patterns at neighbourhood scale.

NDVI map — Amsterdam. Green: high vegetation density. Red/brown: built-up and impervious surfaces.

What is NDVI?

NDVI is a widely used index derived from the ratio of near-infrared (NIR) and red bands in multispectral imagery. Healthy vegetation strongly reflects NIR while absorbing red light; the resulting index value ranges from -1 to +1, where values above ~0.3 indicate active vegetation cover.

Methodology

  • Acquired cloud-free multispectral satellite imagery covering the Amsterdam municipality
  • Calculated NDVI using the standard formula: (NIR − Red) / (NIR + Red)
  • Classified the NDVI surface into vegetation density categories
  • Produced a cartographic output highlighting green space distribution relative to built-up areas

Observations

  • Vondelpark, Amsterdamse Bos, and canal-side greenery register the highest NDVI values
  • Dense inner-city areas show markedly lower vegetation cover
  • Canal surfaces appear as water bodies (negative NDVI), forming a distinctive linear pattern
  • Analysis provides a baseline for monitoring seasonal or long-term urban greening changes

Tools

  • ArcGIS Pro — raster band math, classification, cartographic output
  • Spatial Analyst — raster calculator for NDVI derivation
  • Multispectral satellite imagery (Sentinel-2 / Landsat)
