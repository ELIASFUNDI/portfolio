Ruth: Survey Office Automation

2024  ·  Office Automation Tool  ·  Python · Streamlit · Ollama / LLaMA · python-docx

Ruth is a Streamlit-based desktop application that automates the generation of official survey correspondence at the Mbeere North Sub-County Survey Office. What previously took a surveyor 15–20 minutes per letter (manually filling in parcel details, legal references, and signatory blocks). Now it takes seconds.

Survey office automation: generating official correspondence from parcel records. Illustrative; real letter content withheld.

The Problem

The survey office issues dozens of formal letters weekly: ownership queries, boundary notifications, encroachment notices, and completion certificates. Each letter follows a fixed government template but requires accurate insertion of parcel identifiers, land registry references, owner details, and legal clauses. Manual drafting was slow, error-prone, and created a backlog that delayed land transactions.

How Ruth Works

  • User selects the letter type from a dropdown (ownership query, boundary notice, etc.)
  • Enters parcel reference number and any variable fields
  • Ruth pulls relevant data, fills the government-standard template using python-docx
  • An integrated Ollama / LLaMA model assists with drafting free-text paragraphs when required
  • Output is a properly formatted .docx ready for printing and official stamp
  • Batch mode: generate letters for a list of parcels from a CSV in one click

Impact

  • Letter generation time reduced from ~15 minutes to under 30 seconds per letter
  • Eliminated common errors in legal reference numbers and land registry citations
  • Batch mode allows 47 letters to be produced in under 4 seconds
  • Adopted by office staff with no programming background after a single demonstration

Privacy Note

Ruth handles official government land records. No real parcel data, owner information, or letter content is displayed here. All screenshots and illustrations use sanitised or synthetic data only.

Stack

  • Python: core application logic
  • Streamlit: desktop UI (runs locally)
  • python-docx: Word document generation from templates
  • Ollama / LLaMA: local language model for free-text drafting
  • pandas: CSV batch input handling
